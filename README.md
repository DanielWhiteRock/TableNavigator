# Tables Navigator

An angular + NodeJs tables navigator 

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install.

```bash
yarn install
```

## Usage
#### Run Angular ( Front End )
#### cd packages/angular
```javascript
yarn run start
```

#### Run NodeJs ( Back End )
#### cd packages/api/service
```javascript
yarn run start
```
